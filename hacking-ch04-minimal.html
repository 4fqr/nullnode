Chapter 04: Advanced Linux Commands

Mastering Command-Line Power Tools

Beyond the Basics

You've learned the fundamentals. Now it's time to level up with advanced commands that separate beginners from professionals. These tools will make you faster, more efficient, and deadlier during penetration tests.

Every command in this chapter is explained from scratch - no assumptions, no skipped steps.

Topic 1: Text Processing Masters

grep - Global Regular Expression Print

Search for patterns in files. Essential for finding specific information in logs, configs, or command output. 

# Basic search
grep "password" file.txt

# Case-insensitive
grep -i "PASSWORD" file.txt

# Recursive (search all files in directory)
grep -r "admin" /etc/

# Show line numbers
grep -n "error" log.txt

# Invert match (show lines NOT containing pattern)
grep -v "success" log.txt

# Multiple patterns
grep -e "error" -e "fail" log.txt

# Count matches
grep -c "login" log.txt

awk - Pattern Scanning and Processing

Process text column by column. Perfect for parsing logs and extracting specific fields. 

# Print first column
awk '{print $1}' file.txt

# Print multiple columns
awk '{print $1, $3}' file.txt

# With custom delimiter
awk -F: '{print $1}' /etc/passwd

# Conditional processing
awk '$3 > 100 {print $1}' file.txt

# Sum numbers in column
awk '{sum+=$1} END {print sum}' numbers.txt

sed - Stream Editor

Edit text streams without opening files. Great for find-and-replace operations. 

# Replace first occurrence
sed 's/old/new/' file.txt

# Replace all occurrences
sed 's/old/new/g' file.txt

# Delete lines containing pattern
sed '/pattern/d' file.txt

# Print specific line
sed -n '5p' file.txt

# Print range of lines
sed -n '1,10p' file.txt

# In-place editing
sed -i 's/old/new/g' file.txt

Topic 2: Advanced File Operations

find - Locate Files

# Find by name
find / -name "config.php"

# Case-insensitive
find / -iname "Config.PHP"

# Find by type (f=file, d=directory)
find /var -type f

# Find by size
find / -size +100M              # Larger than 100MB
find / -size -10k               # Smaller than 10KB

# Find by permissions
find / -perm 777
find / -perm -4000              # SUID files

# Find by modification time
find / -mtime -7                # Modified in last 7 days

# Execute command on results
find / -name "*.log" -exec rm {} \;

tar - Archive Files

# Create archive
tar -cvf archive.tar files/

# Create compressed archive (gzip)
tar -czvf archive.tar.gz files/

# Extract archive
tar -xvf archive.tar

# Extract compressed archive
tar -xzvf archive.tar.gz

# List archive contents
tar -tvf archive.tar

# Extract to specific directory
tar -xzvf archive.tar.gz -C /destination/

rsync - Sync Files

# Basic sync
rsync source/ destination/

# With archive mode (preserve permissions, timestamps)
rsync -av source/ destination/

# With progress and compression
rsync -avz --progress source/ destination/

# Remote sync over SSH
rsync -avz source/ user@server:/path/

# Delete files in destination not in source
rsync -av --delete source/ destination/

Topic 3: System Monitoring

df - Disk Space

# Show disk space
df -h                   # Human-readable

# Show specific filesystem
df -h /home

# Show inodes
df -i

du - Directory Usage

# Show directory size
du -sh /var/log

# Show all subdirectories
du -h /var/log

# Sort by size
du -h /var | sort -h

# Top 10 largest
du -h / | sort -rh | head -n 10

lsof - List Open Files

# All open files
lsof

# Files opened by user
lsof -u alice

# Files opened by process
lsof -p 1234

# Network connections
lsof -i
lsof -i :80              # Port 80 specifically

# Who's using a file
lsof /var/log/syslog

Topic 4: Advanced Networking

netcat - TCP/UDP Swiss Army Knife

# Listen on port
nc -lvp 4444

# Connect to port
nc target.com 80

# Transfer file
# Receiver:
nc -lvp 4444 > received_file
# Sender:
nc target_ip 4444 < file_to_send

# Port scanning
nc -zv target.com 20-80

# Reverse shell (listener)
nc -lvp 4444 -e /bin/bash

# Bind shell (target)
nc -lvp 4444 -e /bin/bash

tcpdump - Packet Capture

# Capture on interface
sudo tcpdump -i eth0

# Save to file
sudo tcpdump -i eth0 -w capture.pcap

# Filter by port
sudo tcpdump -i eth0 port 80

# Filter by host
sudo tcpdump -i eth0 host 192.168.1.100

# Filter by protocol
sudo tcpdump -i eth0 icmp

# Verbose output
sudo tcpdump -i eth0 -vv

curl - Transfer Data

# GET request
curl http://example.com

# Save output
curl -o page.html http://example.com

# Follow redirects
curl -L http://example.com

# POST request
curl -X POST -d "user=alice&pass=123" http://example.com/login

# Custom headers
curl -H "User-Agent: Mozilla" http://example.com

# Authentication
curl -u username:password http://example.com

# View headers
curl -I http://example.com

Topic 5: Advanced Process Control

screen - Terminal Multiplexer

# Start new session
screen

# Detach from session (Ctrl+A, then D)

# List sessions
screen -ls

# Reattach to session
screen -r

# Create named session
screen -S mysession

# Reattach to named session
screen -r mysession

# Kill session
screen -X -S mysession quit

tmux - Terminal Multiplexer (Better)

# Start new session
tmux

# Detach (Ctrl+B, then D)

# List sessions
tmux ls

# Attach to session
tmux attach

# Create named session
tmux new -s mysession

# Split panes
Ctrl+B, then "       # Horizontal split
Ctrl+B, then %       # Vertical split

# Navigate panes
Ctrl+B, then arrow keys

These advanced commands are your power tools. Practice combining them with pipes and redirects to create powerful one-liners.
