Web Development Fundamentals

Build websites from scratch using HTML, CSS, and JavaScript - the three languages that power the entire internet.

Every website you visit - Google, Facebook, YouTube, this very page - is built with three core technologies: HTML for structure, CSS for style, and JavaScript for behavior. Master these three, and you can build anything on the web.

HTML is the foundation and walls - the structure.

CSS is the paint, furniture, decorations - the design.

JavaScript is the electricity, plumbing, appliances - what makes it functional.

HTML: The Structure

HTML (HyperText Markup Language) defines the content and structure of web pages using tags.

Basic HTML Document

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Website</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first webpage.</p> </body> </html>

Common HTML Tags

Headings & Text

<h1>Heading 1</h1> <p>Paragraph</p> <strong>Bold</strong> <em>Italic</em>

Links & Images

<a href="url">Link</a> <img src="pic.jpg">

CSS: The Style

CSS (Cascading Style Sheets) makes your HTML look beautiful.

/* Select and style paragraphs */ p { color: white; font-size: 16px; } /* Style buttons */ .button { background: #007bff; padding: 10px 20px; border-radius: 5px; }

JavaScript: The Behavior

JavaScript makes your pages interactive and dynamic.

// Change content when button is clicked document.getElementById('myButton').addEventListener('click', () => { alert('Button clicked!'); });

DOM Manipulation

The DOM (Document Object Model) lets JavaScript interact with HTML elements.

// Select and modify elements const element = document.querySelector('#myDiv'); element.textContent = 'New content!'; element.style.color = 'red';
